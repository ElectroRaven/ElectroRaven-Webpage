# Maximilian Grill Portfolio

This project is a React-based portfolio website built with Vite.

## Prerequisites

- Node.js installed on your machine.

## Setup & Run

1.  **Install Dependencies**:
    ```bash
    cd portfolio-site
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Customization

-   **Social Links**: Identify the placeholders in `src/components/Contact.jsx` and `src/components/Projects.jsx` and replace them with your actual URLs.
-   **Images**: Add your images to `src/assets` and update the references in the components.
